# Day 04 Interview Questions

## What is a condition?

A condition is an expression that evaluates to either true or false.

---

## What are the possible outputs of a condition?

- true
- false

---

## Difference between if and if-else

if:
Executes code only when condition is true.

if-else:
Handles both true and false outcomes.

---

## Difference between if-else and else-if ladder

if-else:
Two possible outcomes.

else-if ladder:
Multiple possible outcomes.

---

## What is nested if?

A nested if is an if statement inside another if statement.

Both conditions must be true before executing the inner block.
