# Day 03 Interview Questions

## 1. What is Input?

Input is the data provided by the user to a program.

Example:

Enter Age:
23

Here 23 is the input.

---

## 2. What is Output?

Output is the information displayed by the program to the user.

Example:

Your age is 23

---

## 3. What is Scanner?

Scanner is a Java class used to take input from the keyboard.

---

## 4. Why do we use Scanner?

Scanner allows users to provide data to a program instead of using fixed values.

---

## 5. Explain:

Scanner input = new Scanner(System.in);

It creates a Scanner named input that reads data from the keyboard.

---

## 6. Important Scanner Methods

nextInt()
→ Integer Input

nextLine()
→ String Input

nextDouble()
→ Decimal Input

nextBoolean()
→ Boolean Input

---

## 7. Why are meaningful variable names important?

Meaningful variable names improve readability and maintainability.

Example:

studentName

is better than

a

because it clearly explains the purpose of the variable.
