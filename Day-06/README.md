## Key Learnings

### while Loop

Used when the number of iterations is not known in advance.

Example:

```java
while(condition)
{
    // code
}
```

---

### Infinite Loop

An infinite loop occurs when the loop condition never becomes false.

Example:

```java
while(true)
{
    System.out.println("Running");
}
```

---

### do-while Loop

A do-while loop executes at least one time before checking the condition.

Example:

```java
do
{
    // code
}
while(condition);
```

---

### break Statement

Used to immediately terminate a loop.

Example:

```java
if(i == 5)
{
    break;
}
```

---

### continue Statement

Used to skip the current iteration and move to the next iteration.

Example:

```java
if(i == 5)
{
    continue;
}
```
