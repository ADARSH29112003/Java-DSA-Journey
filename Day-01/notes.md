# Day 01 Notes - Introduction to Java & Variables

## What is Java?

Java is a programming language used to build software such as web applications, mobile apps, enterprise software, and backend systems.

---

## Why Learn Java?

- Object-Oriented Programming Language
- Platform Independent (Write Once, Run Anywhere)
- Widely used in the software industry
- Strong community support
- Used by many large companies

---

## How Java Works

Java Code (.java)
        ↓
Compiler (javac)
        ↓
Bytecode (.class)
        ↓
JVM
        ↓
Machine Code
        ↓
Program Runs

---

## What is a Variable?

A variable is a named memory location that stores a value.

Example:

int age = 23;

Here:
- int → Data Type
- age → Variable Name
- 23 → Value

---

## Data Types Learned

int
- Stores whole numbers
- Example: 23

double
- Stores decimal numbers
- Example: 1.82

char
- Stores a single character
- Example: 'A'

String
- Stores text
- Example: "Adarsh"

boolean
- Stores true or false
- Example: true
